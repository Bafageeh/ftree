# شجرة النسب الشريف

منصة عربية لتوثيق شجرة النسب وعرض السلاسل التي يثبت اتصالها بسيد البشر محمد ﷺ.

## سياسة النسب المنشور

- الجذر المرجعي الوحيد هو `CORE-001`، وهو محمد ﷺ.
- لا يظهر في الشجرة العامة ولا في قائمة الأشخاص أي اسم أو فرع لا يمتد مسار آبائه المسجل حتى محمد ﷺ.
- المسار المتصل يعرض كاملًا من محمد ﷺ إلى الشخص المختار.
- السلسلة المتصلة قد تكون معتمدة بالكامل، أو متصلة مع بقاء بعض عقدها بانتظار مراجعة المشرف.
- البيانات غير المتصلة تُستبعد من النشر العام، ولا تعود للعرض إلا بعد إثبات الصلة وربطها بالجذر النبوي.
- لا يُنشئ النظام اسمًا أو علاقة نسب غير موجودة في المصادر.

## Laravel API

الروابط الأساسية المنشورة:

```text
GET /api/v1/health
GET /api/v1/stats
GET /api/v1/people
GET /api/v1/people?search=أحمد
GET /api/v1/people?lineage_status=confirmed
GET /api/v1/people?lineage_status=pending
GET /api/v1/people/{id}
GET /api/v1/people/{id}/lineage
```

قواعد الاستجابة:

- `/api/v1/people` يعيد الأسماء المتصلة بمحمد ﷺ فقط.
- طلب تفاصيل أو مسار اسم غير متصل يعيد `404` ولا ينشر السلسلة.
- `/api/v1/stats` يحسب الأسماء المتصلة فقط، وتكون قيمة `disconnected_from_prophet` مساوية للصفر.
- أُلغي مسار `/api/v1/lineage-gaps` من النشر العام.

## الواجهة العامة

```text
GET /lineage
GET /lineage/{person}
```

تعرض الواجهة:

- البحث في الأسماء المتصلة بالنبي ﷺ فقط.
- فصل السلاسل المعتمدة عن السلاسل المتصلة التي تحتاج مراجعة.
- عرض الترابط كاملًا من محمد ﷺ إلى الشخص المختار.
- إخفاء أي اسم أو فرع لا يصل إلى الجذر النبوي.

## النشر

تستخدم النسخة الحالية مهمة GitHub Actions:

```text
Deploy Shajara API
```

تنشر Laravel API والواجهة العامة فقط، وتتحقق آليًا من أن إجمالي الأسماء المنشورة يساوي عدد المتصلين بمحمد ﷺ، ولا تشغّل Expo.

## تنبيه التوثيق

الاتصال التقني في قاعدة البيانات لا يعني وحده الاعتماد العلمي النهائي. كل اسم وعلاقة يحتاجان إلى مصدر موثق واعتماد المشرف قبل وصف السلسلة بأنها معتمدة بالكامل.
