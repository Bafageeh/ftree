# شجرة النسب الشريف

مشروع عربي لتوثيق وعرض شجرة النسب بصورة تفاعلية، مع البحث في الأسماء، إظهار مسار النسب، وتحديد درجة توثيق كل معلومة.

## النسخة الأولى

- واجهة عربية كاملة واتجاه من اليمين إلى اليسار.
- عرض تفاعلي للسلسلة الرئيسية.
- البحث بالاسم أو اللقب.
- فتح بطاقة الشخص وعرض الأب والأبناء ومستوى التوثيق.
- تمييز البيانات المؤكدة، قيد المراجعة، وغير المقروءة.
- نشر تلقائي إلى السيرفر عبر GitHub Actions وبيئة `production`.

## هيكل المشروع

```text
.
├── index.html
├── assets/
│   ├── app.js
│   └── styles.css
├── data/
│   └── people.js
└── .github/workflows/deploy.yml
```

## ملاحظة مهمة

الأسماء الموجودة حاليًا بيانات تأسيسية فقط، ولا تُعد اعتمادًا نهائيًا للنسب. يجب مراجعة كل اسم وربطه بمصدر أو وثيقة قبل تحويل حالته إلى «موثق».
